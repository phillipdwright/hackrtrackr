About This Project

HackrTrackr is a project developed by Greg Karpov, David Granas, and Phil Wright as our final Demo-day project for the Advanced Python Programming Course we completed with Rmotr.com.


HackrTracker uses the Hacker News Search API built by Algolia to source the job posts and the data behind the figures displayed on the site. The trend data uses historical information sourced from the Hacker News Search API. The project also relies on Glassdoor's Companies API for company rating information and for links to Glassdoor's company overview resources, like company reviews, .

If you have any ideas for improvements to the project, drop us a line on GitHub, or send us a pull request!
